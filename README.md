
# SocialConnect

## 📌 Sobre o projeto

O SocialConnect é uma plataforma que tem como objetivo conectar pessoas com interesses e objetivos em comum, facilitando a criação de conexões significativas.

O projeto simula um ambiente corporativo de desenvolvimento, aplicando boas práticas de versionamento com Git e GitHub, como uso de branches, pull requests, issues e commits padronizados.

---

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- Git
- GitHub
